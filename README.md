# Some FORTRAN subprograms used in astronomy
A library based on the Technical Note "SOME FORTRAN SUBPROGRAMS USED IN ASTRONOMY" by John A. Rall, Lincoln Laboritory (16 July 1969)[link DTIC](https://apps.dtic.mil/sti/citations/AD0691816)

As a software engineer with love for the FORTRAN programming language and interests in astronomy, I found this technical note and desided to create a repository.
To share some old FORTRAN code and to preserve and use old subroutines. Not becease it's hard and difficult, but becease it's made simple, usefull, and fast.
